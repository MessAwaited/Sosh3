# Практико-ориентированный курс

Учебный проект для школьников на базе центра образования «Точка роста» МБОУ СОШ №3.

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере:

- `http://127.0.0.1:5173`

Если `http://localhost:5173` не открывается, используйте именно `127.0.0.1:5173`.

## Запуск на другом компьютере

Чтобы проект запустился на другом компьютере, нужно:

1. Установить `Node.js` версии LTS вместе с `npm`.
2. Перенести всю папку проекта на другой компьютер.
3. Открыть терминал в папке проекта.
4. Выполнить:

```bash
npm install
npm run dev
```

5. Открыть в браузере:

- `http://127.0.0.1:5173`

Если нужен не режим разработки, а готовая сборка, выполните:

```bash
npm run build
```

После этого итоговые файлы появятся в папке `dist/`.

## Режимы входа

- Ученик: регистрация по ФИО и классу, после одобрения учителем вход по ФИО, классу и паролю.
- Учитель: вход по учётной записи учителя, просмотр заявок, прогресса учеников и соревнования.

## Основные страницы

- `/login` — вход и регистрация.
- `/student` — кабинет ученика.
- `/student/lesson/:id` — страница урока с тестом.
- `/student/competition` — соревнование для ученика.
- `/student/rating` — рейтинг топ-10 учеников.
- `/teacher` — кабинет учителя с учениками и сводкой по классу.
- `/teacher/competition` — соревнование для учителя: заявки, участники, проверка работ и ZIP-архивы.

## Возможности учителя

- Одобрение заявок и выдача пароля ученику.
- Просмотр прогресса по классам.
- Просмотр паролей учеников.
- Сводка по классу: средний прогресс, средний балл и средняя оценка.
- Проверка работ в соревновании с выставлением баллов по критериям.

## Сборка

```bash
npm run build
```

Готовые файлы находятся в папке `dist/`.
